Movie_API

Objective

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.

Description URL Method Query Parameters Request body data format Response body data format
Return a list of ALL movies to the user /movies GET none none JSON object holding all info about movies
Return data about a single movie by title to the user /movies/[title] GET :title none A JSON object holding all the data about a single movie.

Example:
{
title: 'Back to the Future',
director: 'Robert Zemeckis',
genre: 'Sci-Fi'
}
Return data about a genre by name/title /movies/genre/[genre] GET :genre none A JSON object holding all the data about a single genre.

Example:
{
genre: 'Sci-Fi',
description: 'Science fiction (or sci-fi) is a film genre that uses speculative, fictional science-based depictions of phenomena that are not fully accepted by mainstream science, such as extraterrestrial lifeforms, spacecraft, robots, cyborgs, interstellar travel or other technologies.'
}
Return data about a director by name /directors/[name] GET :name none A JSON object holding all the data about a director.

Example:
{
name: 'Robert Zemeckis',
birthdate: '22.07.1957',
deathdate: 'n/a'
}
Allow new users to register /users/register POST none Data of new user for registration.

Example:
{
name: 'John Doe',
birthdate: '22.07.1997',
favorite_genre: 'Sci-Fi'
} 		Text response for successful registration including automatically generated UUID.
Write to console info of new user.

Example:
{
name: 'John Doe',
id: '001',
birthdate: '22.07.1997',
favorite_genre: 'Sci-Fi'
}
Allow users to update their user info /users/update/[id] PUT :id Data of updated user info. Text response indication that the changes have been successfully deployed.
Allow users to add a movie to their list of favorites /users/favourites/add/[id]/[title] POST :id, :title none Text response confirming addition to favourite list.
Allow users to remove a movie from their list of favorites /users/favourites/remove/[id]/[title] DELETE :id, :title none Text response confirming removal to favourite list.
Allow existing users to deregister (showing only a text that a user email has been removed—more on this later) /users/deregister/[id] DELETE :id none Text response confirming deregistration of user.
